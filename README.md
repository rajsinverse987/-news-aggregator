# 📰 News Aggregator with Sentiment Analysis

A Python web app that fetches real-time news and 
analyzes the sentiment of each headline.

## 🚀 Features
- Search any topic (cricket, AI, India, etc.)
- Fetches real news from 70,000+ sources
- Analyzes each headline as Positive/Negative/Neutral
- Shows sentiment score for each article
- Click to read full article

## 🛠️ Built With
- Python
- Flask
- TextBlob (NLP)
- NewsAPI
- HTML & CSS

## ▶️ How to Run
1. Install libraries:
pip install flask requests textblob newsapi-python

2. Run the app:
python app.py

3. Open browser:
http://127.0.0.1:5000
